# 🥝 CodeKiwi

웹 브라우저에서 사용하는 터미널 개발 환경

## 🚀 빠른 시작

```bash
docker-compose up -d --build
```

브라우저에서 http://localhost:8080 열기

**끝!** `~/CodeKiwi` 디렉토리가 자동으로 생성됩니다.

## 🛠️ 구성

- **ttyd** - 웹 터미널
- **tmux** - 세션 유지
- **nginx** - 프록시
- **opencode-ai** - AI 코드 에디터
- **Docker** - 컨테이너화

## 💻 OS 호환

macOS · Linux · Windows (Docker Desktop)
